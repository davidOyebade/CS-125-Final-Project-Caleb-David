üìö GraphQL Schema Documentation

Queries

people: [Person!]!
Returns: List of all people
Retrieves a list of all people from MySQL.
person(personId: Int!): Person
Returns: Person object or null
Retrieves a specific person by their ID.
events: [Event!]!
Returns: List of basic event info
Retrieves a list of all events (basic info only).
eventsWithCounts: [EventWithCustomData!]!
Returns: List of events with check-in counts and custom data
Retrieves all events with check-in counts from Redis and custom data from MongoDB.
event(eventId: Int!): EventWithCustomData
Returns: Event with custom data or null
Retrieves an event with custom field values and check-in count.
smallGroups: [SmallGroup!]!
Returns: List of small groups
Retrieves a list of all small groups.
eventTypes: [EventType!]!
Returns: List of event types with schemas
Retrieves all event types with their custom field schemas.
checkedInStudents(eventId: Int!): CheckedInResponse
Returns: List of checked-in students
Gets all students currently checked in to an event.
checkInCount(eventId: Int!): CheckInCount
Returns: Check-in count for an event
Gets the current count of students checked in to an event.

Mutations

createEventType(eventTypeData: EventTypeCreateInput!): EventType
Returns: Created event type
Creates a new event type with custom fields. Input: {name: String!, customFields: [{fieldName: String!, dataType: String!}]}
checkInStudent(eventId: Int!, studentId: Int!): Boolean
Returns: true if successful
Checks a student into an event.
checkOutStudent(eventId: Int!, studentId: Int!): Boolean
Returns: true if successful
Checks a student out of an event.
